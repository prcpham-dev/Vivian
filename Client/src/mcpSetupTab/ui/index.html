Connecting Vivian to External AI Agents

Use Vivian's structural graph tools directly inside your favorite MCP-compatible AI agent.

1. Copy Configuration

Vivian automatically generates the exact JSON configuration block needed to connect your agent to the FastMCP server.

2. Paste into your Agent

Paste the copied JSON block into your agent's MCP settings configuration file.

  • Cursor IDE: Open Settings (Gear icon) → Features → MCP. Click + Add new MCP server (Type: Command).
  • Antigravity IDE: Open your terminal and paste it into ~/.gemini/antigravity-ide/mcp_settings.json.
  • Claude Desktop: Open Settings → Developer → Edit Config, and paste inside mcpServers.

Magic Prompt Injection: You don't need to write custom rules to tell your agent to use Vivian! The FastMCP server automatically injects a critical instruction that forces the agent to prioritize Vivian's graph tools over its default file-reading abilities.
